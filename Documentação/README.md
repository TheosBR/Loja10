#DOCUMENTAÇÃO DO PROJETO – LOJA10

1. EXPLICAÇÃO DO PROJETO
O projeto Loja10 é um sistema web de e-commerce simples desenvolvido utilizando PHP, MySQL, HTML e CSS. Ele simula uma loja virtual onde é possível visualizar produtos, realizar pedidos via WhatsApp, cadastrar usuários e gerenciar produtos por meio de um acesso administrativo.
O sistema foi desenvolvido com foco em aprendizado prático de desenvolvimento web, integração com banco de dados e manipulação de formulários.
________________________________________
2. OBJETIVO DO SISTEMA
O principal objetivo do sistema é:
•	Criar uma loja virtual funcional
•	Permitir o gerenciamento de produtos
•	Aplicar conceitos de programação web (frontend + backend)
•	Trabalhar com banco de dados MySQL
•	Simular um ambiente real de e-commerce
Além disso, o projeto também tem como objetivo desenvolver habilidades como:
•	Organização de código
•	Segurança básica (uso de sessão para admin)
•	Manipulação de arquivos (upload de imagens)
________________________________________
3. PARA QUE SERVE O SISTEMA
O sistema Loja10 serve para:
Usuários:
•	Visualizar produtos disponíveis
•	Ver imagens, nomes e preços
•	Enviar pedido diretamente para o WhatsApp
•	Criar cadastro no sistema
