DOCUMENTAÇÃO DO PROJETO – LOJA10

1. EXPLICAÇÃO DO PROJETO
O projeto Loja10 é um sistema web de e-commerce simples desenvolvido utilizando PHP, MySQL, HTML e CSS. Ele simula uma loja virtual onde é possível visualizar produtos, realizar pedidos via WhatsApp, cadastrar usuários e gerenciar produtos por meio de um acesso administrativo.
O sistema foi desenvolvido com foco em aprendizado prático de desenvolvimento web, integração com banco de dados e manipulação de formulários.

2. OBJETIVO DO SISTEMA
O principal objetivo do sistema é:
•	Criar uma loja virtual funcional
•	Permitir o gerenciamento de produtos
•	Aplicar conceitos de programação web (frontend + backend)
•	Trabalhar com banco de dados MySQL
•	Simular um ambiente real de e-commerce

Além disso, o projeto também tem como objetivo desenvolver habilidades como:

•	Organização de código
•	Segurança básica (uso de sessão para admin)
•	Manipulação de arquivos (upload de imagens)

3. PARA QUE SERVE O SISTEMA
O sistema Loja10 serve para:

Usuários:
•	Visualizar produtos disponíveis
•	Ver imagens, nomes e preços
•	Enviar pedido diretamente para o WhatsApp
•	Criar cadastro no sistema

Administrador:
•	Acessar área protegida por senha
•	Adicionar novos produtos
•	Fazer upload de imagens dos produtos
•	Remover produtos da loja

4. FUNCIONALIDADES PRINCIPAIS
•	Listagem de produtos dinâmica (vinda do banco de dados)
•	Botão de compra via WhatsApp
•	Sistema de cadastro de usuários
•	Upload de imagens
•	Exclusão de produtos
•	Área administrativa protegida por senha (sessão PHP)

5. FERRAMENTAS UTILIZADAS

Linguagens
•	PHP: Backend (lógica do sistema)
•	HTML:  Estrutura das páginas
•	CSS: Estilização e layout
•	JavaScript (básico): Alertas e confirmações

Banco de Dados
•	MySQL: Armazenamento de dados (produtos, usuários)
Ambiente de Desenvolvimento
•	XAMPP: Servidor local (Apache + MySQL)
Integrações:
•	WhatsApp API (wa.me): Para envio de pedidos diretamente

6. CONCLUSÃO
O projeto Loja10 é uma aplicação simples, porém completa para fins educacionais, permitindo entender na prática como funciona um sistema web com banco de dados.
Ele pode ser expandido para um sistema mais profissional com:
•	Sistema de login completo
•	Carrinho de compras
•	Painel administrativo avançado
•	Integração com meios de pagamento
